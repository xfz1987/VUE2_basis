v-model.lazy:

懒加载的，当改变文本框时，原始文本不变，当文本框失去焦点时，原始文本才会变化
应用: 收索框，当失去输入完，才向后台发送请求数据

v-model.number:

这样就不需要自己处理将字符串转换成数字格式了，便于计算
